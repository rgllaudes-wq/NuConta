Configurar archivo de datos

NuConta puede guardar y cargar tus datos automáticamente en un archivo Data.json en la carpeta que elijas.

Solo necesitas seleccionarlo una vez. A partir de ahí todo es automático y silencioso.

Requiere Chrome o Edge · Funciona en escritorio y Android

Importar movimientos desde Excel / CSV

Selecciona un archivo .xlsx, .xls o .csv y a continuación podrás emparejar sus columnas con los campos de NuConta.

Haz clic o arrastra el archivo aquí
Excel (.xlsx, .xls) o CSV (.csv)
Emparejar columnas del archivo → campos de NuConta
Campo NuConta Columna del archivo Muestra
ℹ️ Si el archivo incluye una columna de categoría, se intentará emparejar automáticamente con las categorías existentes. Las que no coincidan usarán la categoría por defecto.
